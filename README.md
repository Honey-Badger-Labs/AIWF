# AIWF - AI Workflow Automation

GitHub Actions-driven automation for GCP resource provisioning, OpenClaw, and Slack integration with SustainBot.

## Features

- Infrastructure as Code (Terraform) for GCP
- Automated Linux VM provisioning
- OpenClaw workflow orchestration
- Slack bot integration
- Free LLM models (Ollama + LLaMA 2)

See [docs/SETUP.md](./docs/SETUP.md) for detailed setup instructions.

## Cost Optimization
